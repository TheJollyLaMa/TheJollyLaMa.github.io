Command Action Notes
git status table🍁data table🍁data
git add * table🍒data table🍁data
git commit -m "some message" table🍒data table🍁data
git push origin main table🍒data table🍁data
git rm -r pages views etc remove deleted files by directory for clean commit and push after switiching around a file tree
git log see the logs of past commits
git show HEAD The output of this command will display everything the git log command displays for the HEAD commit, plus all the file changes that were committed.
git checkout HEAD filename.txt will restore the file in your working directory to look exactly as it did when you last made a commit.
git reset HEAD filename.txt git reset --soft HEAD~7
where 7 is the number of commits you want to undo 		This command resets the file in the staging area to be the same as the HEAD commit. It does not discard file changes from the working directory, it just removes them from the staging area.
git reset commit_SHA
git reset git reset 5d69206 		command_SHA is the first 7 characters of the commit's SHA.
HEAD is now set to that previous commit.
The commits that came after the one you reset to are gone.
